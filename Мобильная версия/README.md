# Пост мейкер — мобильная версия

Отдельная mobile-first сборка приложения для подготовки коротких христианских постов. Эта версия собрана специально под телефонный экран: один столбец, крупные кнопки, компактный hero-блок и более удобные модальные окна.

Она живет отдельно в папке `Мобильная версия` и не затрагивает основную веб-версию.

## Что умеет

- вводите тему;
- выбираете место Писания из предложенных карточек;
- получаете готовый короткий пост;
- скачиваете картинку поста в PNG;
- при наличии ключа Gemini фон генерируется через Nano Banana по теме и настроению стиха.

- подбирать 8 подходящих мест Писания по теме;
- строить короткий пост на основе темы и выбранного стиха;
- генерировать пост в протестантской богословской рамке с призывом к действию;
- держать пост в пределах до 2000 символов и добавлять 5 хэштегов в конце;
- переключать язык интерфейса, поста и постера: `русский`, `украинский`, `польский`;
- переключать стиль подачи текста: `мягкий`, `вдохновляющий`, `консервативный`, `современный`, `исторический`;
- перегенерировать текст поста без смены выбранного стиха;
- накладывать стих и тему на готовый постер;
- генерировать AI-фон через Gemini Nano Banana;
- настраивать фон и постер через всплывающее окно:
  - сюжет фона;
  - художественный стиль;
  - типографику;
  - расположение текста;
  - прозрачность текста;
  - силу обводки;
- делать fallback на локальный спокойный фон, если AI временно недоступен;
- генерировать новые варианты фона без смены текста.

## Как запустить

1. Если в корне проекта уже есть `.env`, мобильная версия подхватит его автоматически.
2. Если хотите хранить ключ отдельно, скопируйте [`.env.example`](</Users/kirillsevcenko/Desktop/Пост мейкер/Мобильная версия/.env.example>) в `.env` внутри папки `Мобильная версия`.
3. При желании откройте [Запустить мобильную версию.command](</Users/kirillsevcenko/Desktop/Пост мейкер/Мобильная версия/Запустить мобильную версию.command>).
4. Либо из папки `Мобильная версия` выполните `node server.js`.
5. Откройте [http://127.0.0.1:3001](http://127.0.0.1:3001).
6. Дальше сценарий тот же: язык → тема → место Писания → пост → картинка.

## Настройки модели

- по умолчанию используется `gemini-3.1-flash-image-preview` - это Nano Banana 2;
- если хотите использовать классический Nano Banana, укажите `GEMINI_IMAGE_MODEL=gemini-2.5-flash-image`;
- по умолчанию для текста используется `gemini-2.5-flash`;
- если ключ не задан или API недоступно, приложение использует локальный фон как запасной вариант.
